For this black, gold, silver, and white luxury-tech theme, I would use a design system rather than random colors.

### Color Palette

| Purpose              | Color               | Hex       |
| -------------------- | ------------------- | --------- |
| Primary Background   | Deep Black          | `#0A0A0A` |
| Secondary Background | Rich Charcoal       | `#111827` |
| Premium Gold         | Muted Metallic Gold | `#C9A55A` |
| Highlight Gold       | Soft Gold Glow      | `#E0C27A` |
| Silver               | Brushed Silver      | `#C7CDD6` |
| Pure White           | Typography          | `#F8FAFC` |
| Muted White          | Secondary Text      | `#D1D5DB` |
| Border Gold          | Thin Dividers       | `#9C7A35` |

---

### Gradients

#### Gold Gradient

```css
background: linear-gradient(
  135deg,
  #8C6B2A 0%,
  #C9A55A 50%,
  #E0C27A 100%
);
```

#### Silver Gradient

```css
background: linear-gradient(
  135deg,
  #8B949E 0%,
  #C7CDD6 50%,
  #F3F4F6 100%
);
```

#### Background Gradient

```css
background: linear-gradient(
  180deg,
  #0A0A0A 0%,
  #111827 100%
);
```

### Typography

#### Heading Font

**Cinzel**

```css
font-family: 'Cinzel', serif;
```

Good for:

* Mv Singh
* Section headings

---

#### Body Font

**Inter**

```css
font-family: 'Inter', sans-serif;
```

Good for:

* Services
* Contact information

### Border Radius

```css

```

Used throughout:

* Visiting cards

### Icon Style

* Thin outline icons
* Gold stroke
* No filled icons
* 2px stroke width
* Subtle gold glow only on hover

Think:

* Apple luxury
* Mercedes dashboard
* Premium fintech UI
